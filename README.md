🍱 Bento Grid Layout

A responsive Bento Grid Layout built as part of a Frontend Mentor challenge.
The project focuses on creating a visually rich, multi-card layout using advanced CSS Grid techniques.

🚀 Features

- Complex grid-based layout
- Multiple card sections with different sizes
- Responsive design (mobile → desktop)
- Clean modern UI
- Structured content sections
- Flexible layout using CSS Grid

| Technology       | Purpose                 |
| ---------------- | ----------------------- |
| **HTML5**        | Semantic page structure |
| **CSS3**         | Styling and layout      |
| **CSS Grid**     | Main layout system      |
| **Flexbox**      | Internal alignment      |
| **GitHub Pages** | Deployment              |
